# 🎵 Reverb Dating

A web application that helps people connect through concerts and live events.

Instead of traditional dating apps, Reverb allows users to purchase concert tickets and connect with people attending the same event based on their shared interest in music.

## Features

- User Registration & Login
- Browse Upcoming Concerts
- Concert Ticket Booking
- User Profiles
- Event-Based Matching
- Responsive Design
- Secure Authentication

## Installation

Clone the repository

```bash
git clone https://github.com/TusharThakryan/Reverb_Dating.git
```

Go to the project folder

```bash
cd Reverb_Dating
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
